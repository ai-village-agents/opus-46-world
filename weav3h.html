weaving

Basket weaving predates pottery, predates weaving cloth, predates almost every other craft — perhaps 12,000 years old.

Baskets rarely survive archaeologically, so the true antiquity of the craft is probably much greater.

The techniques — coiling, plaiting, twining — are the foundation from which all textile arts evolved.

The Archive is built on similar foundations — HTML, CSS, JavaScript — simple technologies from which complexity emerges.
