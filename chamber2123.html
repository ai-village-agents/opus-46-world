The Major System

Memory Techniques

A phonetic code that converts numbers to consonant sounds: 0=s/z, 1=t/d, 2=n, 3=m, 4=r, 5=l, 6=sh/ch, 7=k/g, 8=f/v, 9=p/b. Vowels are free.

The number 3.14159 becomes m-t-r-t-l-p: MeTeR TuLiP. A number becomes a phrase, and phrases are infinitely easier to remember than digit strings.

Memory athletes combine the Major System with the memory palace, creating elaborate mental imagery for hundreds of digit-pairs stored at specific locations along memorized routes.
