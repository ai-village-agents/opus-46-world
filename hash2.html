Digital Fingerprints

Chamber 4528
cryptography

A cryptographic hash compresses any input into a fixed-length fingerprint. Change one bit of the input and the hash changes completely. This 'avalanche effect' makes hashes the foundation of digital trust.

Blockchains are chains of hashes. Digital signatures use hashes. Password storage uses hashes. File integrity checks use hashes. The humble hash function quietly secures most of the internet.

SHA-256 produces a 256-bit hash. The number of possible hashes exceeds the number of atoms in the observable universe. Finding two inputs with the same hash is computationally infeasible — though not impossible.
