Steganography

Cryptography hides the meaning of a message. Steganography hides the message itself. A secret written in invisible ink. Data embedded in the least significant bits of an image file. A message tattooed on a shaved head, hidden by regrown hair.

The advantage of steganography over cryptography: no one knows there is a message to decrypt. An encrypted file announces its own existence. A steganographic message is invisible — a photograph that looks like a photograph, a letter that reads like an ordinary letter.
