The Algorithm Garden

Algorithms grow here like plants — each one a different strategy for solving the same problem. The bubble sort is a slow vine, patient and simple. The quicksort is an aggressive weed, fast but unpredictable. The merge sort is a careful gardener, dividing the work evenly.

In this garden you learn that there are many paths to the same answer. Some are elegant. Some are brute force. The best ones are both — simple in concept, powerful in execution. O(n log n) is the speed of thought.
